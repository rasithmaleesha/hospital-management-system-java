# 🏥 Hospital Management System

> A desktop-based Hospital Management System built using **Java Swing**, **NetBeans**, and **MySQL**.

---

## 📌 Overview

The Hospital Management System is a desktop application designed to simplify hospital administration by managing patient records, diagnosis details, and hospital information through an easy-to-use graphical interface.

---

## ✨ Features

- 🔐 Secure Admin Login
- 👤 Patient Registration
- 🩺 Add Diagnosis Records
- 📋 View Patient History
- ✏️ Update Patient Details
- 🏥 Hospital Information Management
- 💾 MySQL Database Integration
- 🖥️ User-Friendly Java Swing Interface

## 🚀 Installation

1. Clone this repository

```bash
git clone https://github.com/rasithmaleesha/hospital-management-system-java.git
```

2. Open the project using NetBeans.

3. Create a MySQL database.

4. Import the SQL file.

5. Update the database username and password.

6. Run the project.
